# Agentic arXiv Research Scanner

## Agentic AI Research Intelligence Platform for AI/ML + Quant Finance Papers

An end-to-end autonomous research intelligence system that scans arXiv, summarizes papers using local LLMs, performs semantic RAG retrieval, ingests full PDFs, analyzes research trends, clusters topics, evaluates RAG quality, and delivers reports through Streamlit, email, Slack, and autonomous scheduling.

---

## Features

- Agentic AI workflow orchestration
- LangGraph-based pipelines
- arXiv ingestion
- Ollama local LLM summarization
- SQLite persistence
- ChromaDB vector storage
- Full-paper PDF ingestion
- Chunked RAG retrieval
- Streamlit dashboard
- Research QA chatbot
- Trend analytics
- Topic clustering
- RAG evaluation metrics
- Autonomous scheduler
- Email digest
- Slack notifications
- Workflow observability
- GitHub Actions CI
